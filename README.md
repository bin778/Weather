## 프로젝트 소개
* OpenWeatherMap API를 활용한 날씨 웹 사이트입니다.
* 세계 도시별 실시간 날씨 정보(기온, 강수확률, 습도, 풍속, 날씨) 및 5일간 날씨 예보를 확인할 수 있습니다.
* 반응형 웹을 구현하여 PC 뿐만 아니라 태블릿, 모바일에서도 원활하게 이용할 수 있습니다.

## 개발 인원
* 최영빈(개인 토이 프로젝트)
